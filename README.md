PandaTrip
=======

Conception de l'API Rest de PandaTrip.

### Les dépendances :

Télécharger Composer :
[https://getcomposer.org/download/](https://getcomposer.org/download/)

Installer les librairies :
```
composer.phar install
```

### Lancer le serveur :

Run :
```
php -S 127.0.0.1:8000 -t public/
```
